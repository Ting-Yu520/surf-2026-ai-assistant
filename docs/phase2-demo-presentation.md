## ① 开场：研究问题到项目定位（3 分钟）

### Step 1 — 用具体瞬间抓住注意力（30 秒）

> "世界杯小组赛日本对荷兰，第 88 分钟，日本队身高全面劣势，硬是在最后时刻靠角球扳平。非常震撼。但如果你完全不懂足球——你看到的只是一群人挤在一起，球弹了一下，进了。你甚至不知道刚才发生了什么。"

### Step 2 — 点出研究空白（45 秒）

> "AI 现在已经能看懂足球——DeepMind 的 TacticAI 预测角球战术有 90% 的准确率。但输出是球员坐标和概率，只有教练能看懂。这就是 **体育 AI 分析的'最后一公里'**——有顶级的战术 AI，但没有一个 **面向普通人的翻译层**。"

### Step 3 — 给出你的答案（45 秒）

> "我的项目就是做这个翻译层——用 LLM 把专业战术数据翻译成零基础观众能听懂、觉得有趣的故事。这个框架叫 **Generative HCI for Sports Analytics**：用生成式 AI 做体育分析的科普翻译。"

### Step 4 — 过渡到 v2 进展（1 分钟）

> "上次的 v1 用了 3 个虚构场景，证明了想法做得通。过去三周我把完整管线跑通了——从真实比赛视频到 AI 解说视频，全自动。现在给您看从 v1 到 v2 的具体进展。"

---

## ② v2 进展 + Live Demo（10 分钟）

### Stage 1 — 展示数据集（2 分钟）

移动至 "数据已经不再是编的了"

> "三周前是 3 个虚构场景。这次我从世界杯小组赛第一天（6/11）开始每天手动收录角球进球，到现在 **21 个真实角球**，覆盖 **16 场不同比赛**。每条数据都是真实的，来源可追溯——央视体育、懂球帝——其中 21 个有原始视频。v1 最大的短板——'数据太假'——已经解决了。"

### Stage 2 — 展示管线功能（3 分钟）

指着 Streamlit 流程图说：

> "管线四步全自动：① 央视报道 → ② DeepSeek LLM 双口相声生成 → ③ 微软 Edge TTS 中文配音 → ④ 合成到原视频上。"

打开一个有音频/文本的条目。朗读一段解说，例如：

> "这就像一米七的篮球小个子，硬是从两米的中锋头上抢下篮板还投进了压哨球——这就是'大卫战胜了歌利亚'。"

**关键提示：**
- AI 不凭空编造——每句话基于央视真实报道
- LLM 只负责"翻译"报道的语言风格，事实不会错
- 系统可规模化——同一管线可处理任意数据

### Stage 3 — 播放 AI 解说视频（5 分钟）

随机打开一个已有视频的条目，播放。

**引导语：**

> "您注意三点：第一，声音是两个人对话不是独白；第二，红蓝边框和字幕跟声音同步；第三，整个过程没有人工介入。"

**视频播完后总结：**

> "这就是 v2 的成果——从一条央视新闻出发，全自动生成一档有声有画的 AI 科普短视频。管线从数据输入到视频输出，无需人手触碰键盘，具备了规模化生产的潜力。"

### 两个候选方向（递出去让导师选）

- **方向 A：系统深化** —— 接入 VLM（Gemini），替代手工文章输入，实现"给比赛视频 → 全自动出解说"的完整原型。
- **方向 B：用户预研** —— SURF 剩余时间不做开发，找 5-10 个不看足球的同学做一次预实验，验证"AI 解说比传统解说更容易理解"这个假设。

## 附录：项目关键数字（提醒自己用的）

| 指标 | v1 | v2 |
|------|----|----|
| 数据场景 | 3 个虚构 | 21 个真实世界杯角球 |
| 数据来源 | 无 | 央视体育 / 懂球帝 / 新浪体育 |
| 视频素材 | 无 | 21 个原始视频 |
| AI 输出 | 纯文本 | 文本 + 音频 + 视频 |
| 管线步数 | JSON → 文本（2 步） | 文章 → LLM → TTS → 视频（4 步自动） |

### 技术栈速记

- **LLM**：DeepSeek V4 Pro（Anthropic 兼容接口）
- **TTS**：微软 Edge TTS（`zh-CN-XiaoxiaoNeural`，`+10%` 语速）
- **视频**：ffmpeg（zoompan 推拉 + drawtext 字幕 + drawbox 红/蓝边框）
- **UI**：Streamlit
- **Prompt 核心 IP**：禁止术语 / 强制生活类比 / 数据约束防幻觉 / 角色绑定（懂哥 A 红 + 小白 B 蓝）

### 若导师问起竞品（备用回答）

> "TacticAI 做角球预测，输出给教练看。SoccerMaster 做视频理解，输出给球迷看。**没人做'给完全不懂足球的人看的科普解说'。** 这就是我的位置——AI 体育分析的最后一公里。不是预测更准，而是让更多人能懂。"
